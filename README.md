# DataPlot (数据绘图分析与格式转换工具)

DataPlot 是一款基于 Python Tkinter 和 Matplotlib 开发的多功能数据可视化与后处理工具。它专为海量实验数据（如仿真热流、环境温度采集、电池充放电测试等）的快速清洗、交互绘图、对比分析和格式转换而设计。

## 🌟 核心功能详解

### 1. 多维度交互式绘图系统

* **多轴同屏绘制**：支持多达 3 个独立的 Y 轴坐标系（Y1、Y2、Y3），各轴支持独立的线型（实线、虚线、点划线）与颜色方案。
* **专业级图表样式**：统一的坐标轴边框加粗与刻度朝内设计。坐标轴字体、字号、图例位置、图例列数等均可通过界面实时调整，并持久化保存。
* **标准交互操作**：Matplotlib 原生支持的拖拽缩放、局部放大、自适应复位及坐标数值悬停提示。

### 2. 电池测试数据与微分分析（BATTERY 面板）

* **自动通道映射**：载入电池文件后自动智能识别“循环号”、“工步号”、“时间”、“电压”和“电流”等核心通道列。
* **充放电容量积分**：在不同循环段内，支持基于时间步长（TimeStep）对电流进行时间积分，并支持设定“CC极性”（正/负）以自动翻转电流方向获取充放电累计容量。
* **相对工步时间重算**：在筛选特定工步时，自动将进入该工步的初始时刻重调为 `0`，方便不同循环段的数据横向以相对工步时间进行对比。
* **微分容量（dQ/dV）与微分电压（dV/dQ）计算**：
  * 自动过滤充放电中电流为 `0` 的静置/休息工步，完全消除曲线奇异毛刺与无限大噪声。
  * 内置**移动平均**、**中值滤波**及 **Savitzky-Golay (SG) 高阶滤波**算法。
  * 针对数据点极少的短工步，实现**自适应窗口缩减与安全降级平滑机制**，杜绝滤波导致的程序崩溃。

### 3. 高 DPI 兼容性与紧凑智能布局

* **4K/高分屏自适应**：支持高 DPI 感知，界面文字与控件清晰无模糊。
* **分组框（QGroupBox）紧凑布局**：将侧边栏所有折叠分组框的顶部外边距从 `15px` 优化收缩为 `0px`，整体页面可用高度增加约 **`120px`**，避免频繁上下拖动滚动条。
* **X 轴一列化极简配置栏**：将 X 轴选择与范围限定紧凑地配置在单行布局中，彻底删除了冗余的 X 轴“应用”按钮（统一由 Y 轴下方的**绘制**按钮或 `Enter` 回车键触发图纸刷新），并将 X 轴 `Min` / `Max` 宽度增加到 `50px`，`标题` 输入框拓宽至 `100px`，提高显示辨识度。
* **长列名悬浮气泡提示 (Tooltip)**：X 轴与各通道 ComboBox 下拉选择器完全内置 `Qt.ToolTipRole` 悬浮气泡，鼠标放置于相应项上即可直接显示完整列名，免除超长列名显示不全的困扰。
* **高可视度列选择列表**：收窄了 Y1-Y3 选择列表框条目的垂直间距至 `1px 6px`，大幅增加了同等高度下面板能容纳的列表条数。
* **操作按钮无裁剪宽度设计**：将 Y 轴下方的清除、全选、绘制、保存以及图例行的“重置”按钮全部拓宽至 **`60px`**，防止在各类系统 DPI 和大字体下文字被遮挡裁剪。
* **图例行组件紧贴与位置调换**：在图例垂直位置/水平位置输入框右侧加入 Stretch 拉伸布局消除空白间隙，并调换了“图例列数”与“图例字号”的相对网格位置，归类更合理。

### 4. 极速 CSV 转 Excel (csv2xlsx) 转换器

* **超低内存设计**：采用 Polars 批读取流和 `xlsxwriter` 的常驻物理内存模式（`constant_memory=True`）。对于超 1GB、包含数百万行的 CSV 文件，可在几秒钟内极速转换为 Excel 且不占用电脑系统内存。
* **自动平滑分表**：自动遵循 Excel 单张工作表最大 `1,048,575` 数据行的上限，超限数据自动切分并保存至 `Sheet2`、`Sheet3` 等新工作表中。

### 5. 规范化数据导出与保存

* **清洗重命名列名**：导出保存绘图数据时，自动将数据列头重命名为绘图时所采用的干净图例标签（例如去除电池通道末尾的物理单位括号，或将 FLOEFD 模式中 `"卷芯最小值 温度 1"` 重命名为 `"卷芯最小值"`）。
* **面板文件独立前缀**：保存数据文件不再混淆，自动为各面板分配独立命名的导出文件名：
  * FLOEFD 面板 $\rightarrow$ `FLOEFD_Plot_Data.xlsx`
  * GENERAL 面板 $\rightarrow$ `GENERAL_Plot_Data.xlsx`
  * BATTERY 面板 $\rightarrow$ `BATTERY_Plot_Data.xlsx`
  * 电池循环对比 $\rightarrow$ `BATTERY_Cycle_Compare_Data.xlsx`

## 📖 详细使用步骤指南

### 第一步：准备数据源与转换

* 如果您的数据是大型 CSV 格式，且您的电脑未配置高性能环境：
  1. 打开软件，在顶部选择对应的**文件类型**（如 `BATTERY` 或 `GENERAL`）。
  2. 点击**浏览**按钮，选择您的 CSV 原始数据文件。
  3. 点击**csv2xlsx**按钮。软件将在后台以多线程极速将您的 CSV 转换并输出为同名 `.xlsx` Excel 文件。
  4. 转换成功后，路径输入框中的路径会自动无缝替换为新生成的 `.xlsx` 文件路径。

### 第二步：参数设置与数据载入

1. 根据文件的表头分布，配置导入参数：
   * **起始行（IniRow）**：指定数据列名称（表头）在哪一行（1-indexed，默认 `1`，表示第1行是表头）。
   * **SKIP** / **NULL**：如果文件开头有仿真元数据，可根据需要设置跳过行数与空行数（FLOEFD 默认值为 `3` 和 `8`；GENERAL 和 BATTERY 默认值为 `0`）。
2. 点击**读取**按钮。程序将自动解析工作簿中的工作表名称，若检测到名称包含 "data" 或 "数据" 将自动高亮选中。
3. 待底部状态栏显示“数据处理完成”，即可在界面上看到自动填充的变量列表。

### 第三步：配置变量映射与比例因子 (仅限 BATTERY 面板)

1. 在载入电池文件后，检查**电池数据配置**：
   * 确认下拉框映射是否正确（**循环列**、**工步列**、**时间列**、**电压列**、**电流列**）。
   * **单位比例换算**：如果测试设备导出的电流为 mA、电压为 mV，可在下方输入**电压比例**和**电流比例**为 `0.001`；若已经是 A 和 V，保持默认值 `1` 即可。
2. 配置**工步筛选**与**循环筛选**：
   * 默认两个筛选框都为 `"全部"`。如果只想绘制特定工步（例如“恒流充电”）或特定循环段的数据，从下拉框中选择对应的编号即可。
   * 修改映射或筛选配置后，点击右侧的**应用**按钮，完成数据重新切片与更新。

### 第四步：普通绘图 (非循环对比模式)

1. 确保**启用循环对比**复选框处于**未勾选**状态。
2. 配置绘图轴数据：
   * 在底部的**X轴**下拉框中，选择作为时间或容量的横坐标列（悬停可查看列名全称）。
   * 从下方 **Y1**、**Y2**、**Y3** 三个列表框中多选您要绘制的数据通道。支持滚轮滚动与 `Ctrl` 键多选。
   * （可选）在 Y1-Y3 的 `Min` / `Max` / `标题` 中自定义限值与轴标题。若要恢复图表自适应缩放，只需将 `Min` 和 `Max` 输入框清空。
3. 点击 **绘制** 按钮（或键盘回车），右侧的 Matplotlib 画布将立即输出对应的波形。

### 第五步：启用电池循环对比与 dQ/dV 分析 (仅限 BATTERY 面板)

1. 勾选**启用循环对比**。此时左侧会动态滑出**循环对比配置**面板。
2. 配置对比参数：
   * **对比范围**：输入需要横向比对的循环，例如输入 `1, max, 10` 表示比对从循环 1 开始，每次递增 10 周期，直到最大循环；也可以输入特定的单循环 `2, 5, 10`。
   * **步长(TimeStep)**：输入微分插值的时间跨度（秒），默认 `10` 秒。
   * **CC极性**：正（充电段微分）或 负（放电段微分）。
   * **滤波方式/窗口大小/SG阶数**：选择平滑平坦毛刺的滤波机制。
   * **循环X轴**：支持选择 `容量`、`循环时间`、`Index` 或 `电压` 作为比对图表的横坐标。
3. 选择**比对类型**（单选框）：
   * **常规对比**：将不同循环周期的同一物理量（如电压随容量的变化）绘制在同一张图纸上。
   * **dQ/dV**：绘制微分容量（Ah/V）随电压（V）变化的对比曲线。
   * **dV/dQ**：绘制微分电压（V/Ah）随容量（Ah）变化的对比曲线。
4. 在 **Y1/Y2/Y3 列表框**中勾选需要对比绘制的 Y 轴数据项（注：在 dQ/dV 和 dV/dQ 模式下，Y 轴强制对应微分曲线本身，此时只需确保 Y1 列表中勾选了对应的目标变量即可）。
5. 调整**循环Y轴**：
   * 在“循环Y轴”一行，输入 `Min` 和 `Max` 自定义对比图表的上下限范围（例如 dQ/dV 可以输入 `Min: 0`, `Max: 4`；X轴限值可以使用下方的 X轴 Min/Max）。
   * 点击右侧的 **应用** 按钮（或点击 Y 轴下方的 **绘制** 按钮），一键刷新对比图表。

### 第六步：保存绘图数据

* 确认右侧画布上已经正确绘制出您想要的曲线。
* 点击绘图选项下方的**保存数据**按钮。
  * 如果是普通绘图，会将当前绘制的横纵坐标清洗后的干净数据保存为 `[前缀]_Plot_Data.xlsx`。
